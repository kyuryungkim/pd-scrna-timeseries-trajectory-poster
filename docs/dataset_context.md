# Dataset Context

## Primary Dataset

- GEO accession: GSE204796
- Assay: single-cell RNA-seq
- Context: dopaminergic neuron differentiation in a Parkinson's disease cell-therapy study
- Timepoints of interest: D8, D14, D21, D28, D35

## Framing Used in This Repository

This repository treats the dataset as a public time-course single-cell resource for pseudotime interpretation.

The dataset should be described as:

> a dopaminergic neuron differentiation time-course generated in a Parkinson's disease cell-therapy context

It should not be described as:

> a direct Parkinson's disease progression dataset

## Role of the Original Study

The original study provides the biological and experimental context for the dataset. This e-poster project does not aim to reproduce or validate the original study, and it does not claim new discovery of markers reported in that work.

The narrower goal here is to ask what static scRNA-seq snapshot-derived pseudotime can and cannot tell us.

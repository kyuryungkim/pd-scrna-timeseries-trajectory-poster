# Interpretation Scope

This document defines how pseudotime results should be interpreted in this e-poster project.

## Core Question

> What can pseudotime inferred from static scRNA-seq snapshots tell us, and what can it not tell us?

## What This Analysis Can Support

- Static scRNA-seq snapshots can be ordered by transcriptomic similarity.
- Inferred pseudotime can be compared with known experimental days.
- Marker/module behavior can help interpret early, intermediate, and late cell states.
- A public time-course scRNA-seq dataset can serve as a known-time calibration example for pseudotime interpretation.

## What This Analysis Cannot Prove

- The same cells were tracked over time.
- Pseudotime equals exact chronological time.
- The inferred trajectory proves true lineage tracing.
- The dataset directly captures Parkinson's disease progression.
- This project discovers markers already reported in the original study.
- This project validates or outperforms the original study.

## Preferred Wording

Use:

- static snapshot-derived pseudotime
- transcriptomic ordering
- progression-like cell-state axis
- temporal consistency
- marker/module behavior
- known-time calibration reference

Avoid:

- validated trajectory
- real temporal progression
- lineage tracing
- exact time
- reconstructed Parkinson's disease progression
- novel marker discovery
